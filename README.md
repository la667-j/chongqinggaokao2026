# 重庆物理类 · 高考志愿填报地图 🎓🗺️

> 一个基于 **2023 / 2024 / 2025 三年重庆物理类本科批官方投档数据**的可视化志愿填报工具：在中国地图上按**位次 / 专业 / 院校层次**实时定位可报院校。

**在线访问**：https://la667-j.github.io/chongqinggaokao2026/
（需先在仓库 Settings → Pages 开启；手机 / 电脑均可打开）

---

## ✨ 功能

- **位次区间筛选**：输入你的位次范围（如 `1000 ~ 2000`），地图实时高亮含可报院校的省份。
- **飞线 + 热力高亮**：重庆向匹配省份发出动画飞线，省份颜色越亮代表可报院校越多。
- **多维筛选**：985 / 211 / 双一流 / 普通院校，专业关键词，院校关键词，可叠加。
- **点省份** → 弹出该省院校列表（按最低位次排序，显示位次区间、投档分、匹配专业数）。
- **点院校** → 三年对比（投档线 / 最低位次 / 最高分 / 录取专业数）+ 各年专业明细，符合当前位次的专业自动标记。
- **年份切换**：2023 / 2024 / 2025 任意切换作为基准年。
- **响应式**：桌面三栏布局；手机端地图全屏、筛选收进底部弹层、抽屉与弹窗全屏。
- **纯前端、可离线**：双击 `index.html` 即可运行，ECharts 与地图数据均已本地化。

## 🚀 本地运行

直接用浏览器打开 `index.html`（推荐 Chrome / Edge），无需联网、无需安装。

## 🔧 重新生成数据

更新 Excel 源文件后运行：

```bash
pip install openpyxl
python build_data.py
```

脚本默认读取本地三个 Excel（路径见 `build_data.py` 顶部，可修改），输出 `data.js`。

## 📂 结构

| 文件 | 说明 |
|---|---|
| `index.html` / `styles.css` / `app.js` | 界面与逻辑 |
| `data.js` | 由 Excel 生成的院校 / 专业数据（997 所院校） |
| `build_data.py` | 数据生成脚本 |
| `geo_meta.json` | 省份中心坐标（飞线用） |
| `vendor/` | 本地化的 ECharts 与中国地图 GeoJSON |

## 📊 数据说明

- 院校所在省份由**招生代码前两位**判定（北京 11 / 上海 31 …）；分校按校区实际所在省归类（哈工大威海→山东、哈工大深圳→广东）。
- 「位次」为各专业投档线对应位次，越小=分数越高。
- 层次依据教育部 985/211 名单 + 第二轮双一流名单（2022，147 所）。
- 分数与位次来自重庆市教育考试院官方平行志愿投档表 + 一分一段表。

> ⚠️ 本工具仅供参考，志愿填报请以官方招生计划与权威渠道为准。

## 🛠 技术栈

原生 HTML / CSS / JavaScript + [Apache ECharts](https://echarts.apache.org/)（地理坐标系 / 飞线 / 涟漪散点），Python + openpyxl 处理数据。
