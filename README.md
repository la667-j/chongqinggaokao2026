# 重庆高考 · 志愿填报地图（物理类 / 历史类）🎓🗺️

> 一个基于 **2023 / 2024 / 2025 三年重庆本科批官方投档数据**的可视化志愿填报工具：**物理类、历史类一键切换**，在中国地图上按**位次 / 专业 / 院校层次**实时定位可报院校。

**在线访问**：https://la667-j.github.io/chongqinggaokao2026/
（需先在仓库 Settings → Pages 开启；手机 / 电脑均可打开）

---

## ✨ 功能

- **物理类 / 历史类 切换**：顶栏一键切换科类，地图、统计、年份、列表全部联动刷新。
- **位次区间筛选**：输入你的位次范围（如 `1000 ~ 2000`），地图实时高亮含可报院校的省份。
- **飞线 + 热力高亮**：重庆向匹配省份发出动画飞线，省份颜色越亮代表可报院校越多。
- **查看全部院校**：不点地图也能看——把所有匹配院校**按投档位次从低到高**排成一列（带省份标签），一眼看全。
- **多维筛选**：985 / 211 / 双一流 / 普通院校，专业关键词，院校关键词，可叠加。
- **点省份** → 弹出该省院校列表（按最低位次排序，显示位次区间、投档分、匹配专业数）。
- **点院校** → 三年对比（投档线 / 最低位次 / 最高分 / 录取专业数）+ 各年专业明细，符合当前位次的专业自动标记。
- **年份切换**：2023 / 2024 / 2025 任意切换作为基准年。
- **响应式**：桌面三栏布局；手机端地图全屏、筛选收进底部弹层、抽屉与弹窗全屏。
- **纯前端、可离线**：双击 `index.html` 即可运行，ECharts 与地图数据均已本地化。

## 🚀 本地运行

直接用浏览器打开 `index.html`（推荐 Chrome / Edge），无需联网、无需安装。

## 🔧 重新生成数据

原始资料已随仓库放在 [`source-data/`](source-data/)，直接运行即可复现：

```bash
pip install openpyxl pdfplumber

# 历史类：从官方 PDF + 一分一段表 生成投档明细 xlsx（物理类的 xlsx 已直接提供）
python build_history.py

# 物理类 + 历史类 一起打包成 data.js
python build_data.py
```

`build_data.py` 读取 `source-data/`（物理类）与 `source-data/历史/`（历史类）下的投档明细表，输出**双科类** `data.js`（实测可复现）。

## 📂 结构

| 文件 | 说明 |
|---|---|

## 📊 数据来源

本项目的全部信息均来自 [`source-data/`](source-data/) 内的官方文档，未作任何杜撰。两个科类的资料链路一致：

| 科类 | 录取分来源 | 位次来源 | 整理后文件 |
|---|---|---|---|
| 物理类 | `重庆市本科批-物理-平行志愿.pdf` | `重庆物理类一分一段表.xlsx` | `source-data/重庆20XX物理类.xlsx` |
| 历史类 | `重庆市普通高校招生信息表本科批-历史-平行志愿.pdf` | `重庆历史类一分一段表_20XX.xlsx` | `source-data/历史/重庆20XX历史类.xlsx` |

> **数据来源口径**：①录取分=重庆市教育考试院官方平行志愿投档 PDF；②位次=对应科类《一分一段表》的累计人数；③层次=教育部 985/211 名单 + 第二轮双一流名单(2022，147 所)。

### 加工规则
- 院校所在省份由**招生代码前两位**判定（北京 11 / 上海 31 …）；分校按校区实际所在省归类（哈工大威海→山东、哈工大深圳→广东）。
- 「位次」为各专业投档线对应位次，数值越小=分数越高；筛选「1000~2000」即展示投档位次落在该区间的专业 / 院校。
- 层次（985⊂211⊂双一流）依据教育部名单判定；分校 / 医学部 / 中外合作等归入母体院校。历史类层次沿用物理类同代号院校的判定。
- 历史类投档分来自 PDF（含官方水印，已按字号过滤剔除），位次由投档分匹配一分一段表累计人数得出。
- 历史类特控线/本科线：2023=480/407、2024=506/428、2025=515/438（官方）。
- 重庆已合并本科批，无一本/二本。

> ⚠️ 本工具仅供参考，志愿填报请以重庆市教育考试院官方招生计划与权威渠道为准。
