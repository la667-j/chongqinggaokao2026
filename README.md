# 重庆物理类 · 高考志愿填报地图 🎓🗺️

> 一个基于 **2023 / 2024 / 2025 三年重庆物理类本科批官方投档数据**的可视化志愿填报工具：在中国地图上按**位次 / 专业 / 院校层次**实时定位可报院校。

**在线访问**：https://la667-j.github.io/chongqinggaokao2026/
（需先在仓库 Settings → Pages 开启；手机 / 电脑均可打开）

---

## ✨ 功能

- **位次区间筛选**：输入你的位次范围（如 `1000 ~ 2000`），地图实时高亮含可报院校的省份。
- **飞线 + 热力高亮**：重庆向匹配省份发出动画飞线，省份颜色越亮代表可报院校越多。
- **多维筛选**：985 / 211 / 双一流 / 普通院校，专业关键词，院校关键词，可叠加。
- **点省份** → 弹出该省院校列表（按最低位次排序，显示位次区间、投档分、匹配专业数）。
- **点院校** → 三年对比（投档线 / 最低位次 / 最高分 / 录取专业数）+ 各年专业明细，符合当前位次的专业自动标记。
- **年份切换**：2023 / 2024 / 2025 任意切换作为基准年。
- **响应式**：桌面三栏布局；手机端地图全屏、筛选收进底部弹层、抽屉与弹窗全屏。
- **纯前端、可离线**：双击 `index.html` 即可运行，ECharts 与地图数据均已本地化。

## 🚀 本地运行

直接用浏览器打开 `index.html`（推荐 Chrome / Edge），无需联网、无需安装。

## 🔧 重新生成数据

原始 Excel 已随仓库放在 [`source-data/`](source-data/)，直接运行即可复现：

```bash
pip install openpyxl
python build_data.py
```

脚本会读取 `source-data/` 下的三份投档明细表，输出 `data.js`（实测可逐字节复现）。

## 📂 结构

| 文件 | 说明 |
|---|---|
| `index.html` / `styles.css` / `app.js` | 界面与逻辑 |
| `data.js` | 由 Excel 生成的院校 / 专业数据（997 所院校） |
| `build_data.py` | 数据生成脚本 |
| `geo_meta.json` | 省份中心坐标（飞线用） |
| `lib/` | 本地化的 ECharts 与中国地图 GeoJSON |
| `source-data/` | **原始数据文档**（投档明细表 + 一分一段表，见下） |

## 📊 数据来源

本项目的全部信息均来自 [`source-data/`](source-data/) 内的官方文档，未作任何杜撰：

| 文件 | 内容 | 用途 |
|---|---|---|
| `2023/2024/2025重庆市本科批-物理-平行志愿.pdf` | 重庆市教育考试院官方投档表（原始 PDF） | 录取分（投档分）的原始出处 |
| `2023/2024/2025年重庆市高考物理类一分一段表.xlsx` | 分数 → 该分数段人数 → 累计人数 | 位次（累计人数即位次） |
| `重庆2023/2024/2025物理类.xlsx` | 由以上两类原始资料整理而成：各院校专业录取分 + 位次 + 层次 | 程序直接读取，生成 `data.js` |

每年的投档明细表内含官方口径说明，例如 2023 年原表注明：

> **数据来源**：①录取分=重庆市教育考试院《本科批-物理-平行志愿》PDF；②位次=《2023重庆物理类一分一段表(含加分)》；③层次=教育部 985/211 名单 + 第二轮双一流名单(2022，147 所)。

### 加工规则
- 院校所在省份由**招生代码前两位**判定（北京 11 / 上海 31 …）；分校按校区实际所在省归类（哈工大威海→山东、哈工大深圳→广东）。
- 「位次」为各专业投档线对应位次，数值越小=分数越高；筛选「1000~2000」即展示投档位次落在该区间的专业 / 院校。
- 层次（985⊂211⊂双一流）依据教育部名单判定；分校 / 医学部 / 中外合作等归入母体院校。
- 重庆已合并本科批，无一本/二本。

> ⚠️ 本工具仅供参考，志愿填报请以重庆市教育考试院官方招生计划与权威渠道为准。
