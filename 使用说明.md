# 重庆物理类 · 高考志愿填报地图

基于 2023 / 2024 / 2025 三年重庆物理类本科批**官方投档分 + 位次 + 院校层次**数据的可视化志愿填报工具。

## 如何使用

直接**双击 `index.html`** 用浏览器打开即可（推荐 Chrome / Edge），无需联网、无需安装。

### 功能
- **位次区间**：填写你的位次范围（如 `1000 ~ 2000`），或用快捷预设。地图会高亮含可报院校的省份。
- **飞线 + 高亮**：重庆向每个匹配省份发出飞线箭头，省份颜色越亮代表可报院校越多；圆点大小代表数量。
- **院校层次**：985 / 211 / 双一流 / 普通院校 多选筛选。
- **关键词**：按专业（如「计算机」「临床医学」）或院校（如「理工」）名称筛选。
- **点击省份** → 右侧弹出该省院校列表（按最低位次排序）。
- **点击院校** → 弹窗显示该校 **三年对比**（投档线 / 最低位次 / 最高分 / 录取专业数）+ 各年专业明细表，符合当前位次的专业高亮标「符合」。
- **年份切换**：右上角切换基准年份（默认最新 2025），筛选与地图随之更新。
- `Esc` 关闭弹窗 / 抽屉；地图可滚轮缩放、拖动。

## 手机访问 / 分享给别人

界面已做**手机适配**（竖屏堆叠、地图全屏、筛选收进底部弹层、抽屉与弹窗全屏）。但本工具是一堆本地文件，手机点不开本地文件，要分享给别人需先**部署到网上拿一个公网链接**。

### 方案 A：Netlify Drop（最简单，无需账号/命令）
1. 打开 https://app.netlify.com/drop
2. 把**整个项目文件夹**拖进去（至少包含 `index.html`、`styles.css`、`app.js`、`data.js`、`lib/`）。
3. 几秒后得到一个公网网址（形如 `https://xxx.netlify.app`），把这个链接发给别人，手机/电脑都能开。

### 方案 B：GitHub Pages（链接稳定，适合长期）
1. 新建一个仓库，上传上述文件。
2. 仓库 Settings → Pages → Source 选 `main` 分支根目录，保存。
3. 等一两分钟，得到 `https://你的用户名.github.io/仓库名/` 链接。

> 首次加载约 2.3MB（数据 1.3MB + ECharts 1MB），移动网络下等几秒即可，之后有缓存。
> 运行时只需要 `index.html`、`styles.css`、`app.js`、`data.js`、`lib/`；`build_data.py`、`geo_meta.json`、本说明文件可以不上传。

## 文件结构
| 文件 | 说明 |
|---|---|
| `index.html` | 主界面，双击运行 |
| `styles.css` / `app.js` | 界面样式与逻辑 |
| `data.js` | 由 Excel 生成的院校 / 专业数据（997 所院校） |
| `build_data.py` | 数据生成脚本，更新 Excel 后重新生成 `data.js` |
| `geo_meta.json` | 省份中心坐标（飞线用） |
| `lib/` | 本地化的 ECharts 与中国地图 GeoJSON（离线可用） |

## 重新生成数据
更新 Excel 源文件后，运行：
```
python build_data.py
```
> 依赖：`pip install openpyxl`。脚本默认读取 `C:\Users\admin\Desktop\0\2\` 下的三个 Excel，路径可在 `build_data.py` 顶部修改。

## 数据说明
- 院校所在省份由**招生代码前两位**判定（北京 11 / 上海 31 …），分校按校区实际所在省归类（如哈工大威海→山东、哈工大深圳→广东）。
- 「位次」为各专业投档线对应位次；位次越小=分数越高。筛选「1000~2000」即展示投档位次落在该区间的专业 / 院校。
- 层次依据教育部 985/211 名单 + 第二轮双一流名单（2022，147 所）。
